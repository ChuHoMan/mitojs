## Install

### Web

#### use npm

```bash
$ npm i @mitojs/web
```

### use yarn

```bash
$ yarn add @mitojs/web
```

#### use CDN

```javascript
<script src="https://cdn.jsdelivr.net/npm/@mitojs/web/dist/web.min.js"></script>
```

### wx-mini

#### use npm

```bash

```

### use yarn

#### use CDN

